Навыки

Что я использую в работе

Java Script

Этот язык программирования позволяет оживить все что угодно: слайдеры, окна, подсказки, вкладки, получение данных от сервера и многое другое

Python

Этот язык программирования широко используется в интернет-приложениях, разработке программного обеспечения, часто используется для анализа и работы с данными

PostgreSQL

Это реляционная СУБД, которая будет хранить данные вашего сайта или приложения

SQL

Декларативный язык программирования, применяемый для создания, модификации и управления данными в реляционной базе данных

HTML5

Именно он создает каркас вашего сайта или приложения, а пятая версия позволит мне создавать более SEO-оптимизированную структуру вашего продукта

CSS3

Этот язык стилей позволяет мне создавать абсолютно любой внешний вид вашего сайта или приложения. Все ограничивается только вашей фантазией!

React

React — это декларативная, эффективная и гибкая JavaScript-библиотека для создания пользовательских интерфейсов. Она позволяет мне собирать сложный UI из маленьких изолированных кусочков кода - «компонентов».

VueJS

Этот фреймворк позволяет создавать web-приложения. Мы можем создать максимально интерактивный продукт именно под ваши цели

Vuetify

Это прогрессивный фреймворк, который помогает ускорить веб-разработку, если нет необходимости в персонализированном дизайне.

Jquery

Библиотека Jquery позволит ускорить разработку. Без необходимости интегрировать в проект мы её не будем, но навык работы с ней присутствует

Webpack

Один из самых мощных и гибких сборщиков модулей, благодаря которому мы можем быстро и качественно разрабатывать ваш продукт с использованием различных библиотек и фреймворков.

Babel

Babel - это транспайлер, который помогает нам переписывать код современного стандарта Javascript для поддержки более ранних версий браузеров.

Git

Git — система управления версиями с распределенной архитектурой.

Gulp

Gulp — это таск-менеджер для автоматического выполнения часто используемых задач(например, минификации, тестирования, объединения файлов)...

Sass/Scss

Sass — это метаязык на основе CSS, предназначенный для увеличения уровня абстракции CSS-кода. Используя его, мы упрощаем файлы каскадных таблиц стилей.

Bootstrap

Cвободный набор инструментов для создания сайтов и веб-приложений. Помогает нам разрабатывать продукт быстрее, если нет строгих требований по макету.

JSON

Текстовый формат обмена данными, основанный на JavaScript. Легко читается человеком и машиной.
